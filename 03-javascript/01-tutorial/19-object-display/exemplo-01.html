Displaying Properties in a Loop

Script

        
          // Create an Object
          const person = {
            name: "John",
            age: 30,
            city: "New York"
          };
          
          // Build a Text
          let text = "";
          for (let x in person) {
            text += person[x] + " ";
          };
          
          // Display the Text
          document.getElementById("demo").innerHTML = text;
