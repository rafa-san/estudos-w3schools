Estrutura Básica de um Documento HTML

Código

<!DOCTYPE html>
<html>
<head>
<title>Título da Página</title>
</head>
<body>
<p>Hello, world!</p>
</body>
</html>

Navegador

Hello, world!

  1. A declaração <!DOCTYPE html> define que este é um documento HTML5. Isso ajuda o navegador a exibir o conteúdo da página web corretamente.
  2. O elemento <html> </html> é o elemento raiz de uma página HTML.
  3. O elemento <head> </head> contém meta informações sobre a página HTML.
  4. O elemento <title> </title> define o título da página HTML, que é mostrado na barra de título do navegador ou na guia da página.
  5. O elemento <body> </body> define o corpo do documento e serve como um container para todos os conteúdos visíveis, como títulos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.
  6. O elemento <p> </p> define um parágrafo.

Elementos HTML

Elemento HTML é uma declaração composta por: tag de abertura, conteúdo e tag de fechamento.

Exemplo:

<p>Hello, World!</p>

Elementos HTML vazios

Em alguns casos, o elemento HTML não possui conteúdo e nem a tag de fechamento.

Exemplo:

<br> <!-- Quebra de Linha -->

<hr> <!-- Linha Horizontal -->

Atributos HTML

Os atributos HTML fornecem informações adicionais sobre um elemento HTML.

Todos os elementos HTML podem ter atributos e eles são sempre especificados na tag inicial.

Exemplo:

<a href="https://www.w3schools.com">Visit W3Schools</a>

A tag <a> representa um hyperlink. O atributo href especifica a URL para a qual o link será direcionado.

Comentários em HTML

Comentários não são exibidos no navegador, mas podem ajudar a organizar e documentar seu código-fonte, ao incluir notas, explicações ou lembretes.

Você adiciona um comentário ao seu código HTML da seguinte maneira:

Exemplo:
